result = map2(.x = n_homi$n_unsolved,

``` r
tidy = function(result){
  result %>% 
    broom::tidy()
} 

test_result = map(.x = result, ~tidy(.x)) %>% 
  bind_rows() %>% 
  select(estimate, conf.low, conf.high) %>% 
  mutate(city_state = n_homi$city_state)
```
